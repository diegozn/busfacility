Manifesto Busfacility

   Somos uma empresa que visa melhorar o fluxo de pessoas no transporte público de São Paulo atráves de sensores de presença para calcular em tempo real como os ônibus estão circulando e fornecendo essas informações para as empresas para que tenham melhor controle sobre as suas frotas.

   Com isso buscamos trazer mais conforto e segurança para os usuários do transporte público e uma melhor utilização dos investimentos das empresas.
